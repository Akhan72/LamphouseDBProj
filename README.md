# Mobile Photobooth App also known as its name: Lamphouse Photography Management 

This is built with:
- Flask (Python web framework)
- SQLite (local relational database)
- Chart.js (frontend data visualization)

Features:
- User login and logout with session-based authentication.
- CRUD interface for managing clients.
- Revenue analytics page that aggregates invoice data by package.

To run:
1. Create and activate a virtual environment.
2. Install Flask.
3. Run `python app.py` and open the displayed URL
